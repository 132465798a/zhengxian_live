## ✨ 功能特性

### 🎬 直播源解析
- 支持抖音、B站、斗鱼、虎牙等 50+ 直播平台
- 输入直播间 URL 即可获取直播流地址
- 支持直播录制和保存
- 历史记录保存，方便重复使用
- 支持 HTTP 代理配置

### 💬 弹幕监听
- 多平台实时弹幕抓取（抖音、B站、斗鱼、虎牙）
- 弹幕内容实时显示
- 关键词过滤和用户屏蔽
- 弹幕数量、用户活跃度统计
- 弹幕数据导出为文本文件

### 🔊 TTS 语音播报
- 多引擎支持：系统TTS、火山引擎（120+音色）、百度TTS、阿里云TTS
- 智能播报：弹幕内容、礼物消息、用户进入、关注消息、点赞消息
- 个性化设置：音色选择、语速调节、音量控制、播报间隔
- 预设播报模板，支持自定义

### 📡 RTMP 推流地址解析
- 抖音 RTMP：全自动获取推流地址和串流密钥，多服务器选择
- B站 RTMP：官方接口获取，自动生成推流码，智能推荐最优服务器

### 👁️ 24小时直播监控
- 实时监控直播间开播状态
- 开播自动录制，下播自动停止
- 支持复杂录制工作流
- 录制文件自动上传云存储
- 开播/下播通知推送
- 监控间隔、录制质量、存储路径可配置

### 🎛️ OBS 插件中心
- 自动启动 OBS 软件
- 自动开始推流
- 智能场景切换
- 视频源、音频源管理
- 集成常用 OBS 插件
- 开播自动配置推流参数

### 🌐 多平台弹幕助手
- 支持多个弹幕窗口同时显示
- 透明窗口，可叠加在其他软件上
- 鼠标穿透，锁定状态下可穿透操作
- 弹幕字体、颜色、速度自定义
- 弹幕位置和方向可调整
- 窗口置顶、透明度调节、锁定/解锁

## 🖥️ 系统要求

- Windows 10/11 (64位)
- 4GB 内存
- 500MB 可用存储空间
- 稳定的网络连接

## 🛠️ 技术栈

- 前端框架：Vue 3 + Composition API
- 桌面框架：Electron 28
- 状态管理：Pinia
- UI 组件：TailwindCSS + DaisyUI
- 视频处理：FFmpeg
